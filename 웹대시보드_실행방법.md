# BC-Viz 웹 대시보드 실행 방법

Flask 기반 HTML/CSS/JS 웹 애플리케이션

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

또는 개별 설치:
```bash
pip install flask pandas numpy scikit-learn plotly
```

## 실행 방법

1. 웹 대시보드 서버 시작:
```bash
python web_dashboard.py
```

2. 브라우저에서 접속:
```
http://localhost:5000
```

## 주요 기능

### 1. 다크모드/라이트모드 토글
- 사이드바 상단의 테마 토글 버튼 클릭
- 설정은 브라우저 로컬 스토리지에 저장됩니다

### 2. 데이터 개요
- 전체 샘플 수, 양성/악성 분포
- 진단 분포 차트 (파이 차트, 바 차트)
- 데이터 미리보기 테이블

### 3. 데이터 시각화
- Boxplot: 특징별 분포 및 이상치 확인
- 히스토그램: 진단별 특징 분포 비교
- 특징 선택 가능 (최대 10개)

### 4. 머신러닝 모델
- Random Forest 및 Decision Tree 모델 학습
- 모델 성능 정확도 표시
- Confusion Matrix 시각화
- Feature Importance 차트

### 5. 차원 축소
- PCA (Principal Component Analysis)
  - 2D/3D 시각화
  - 설명된 분산 비율 표시
- t-SNE (t-distributed Stochastic Neighbor Embedding)
  - 2D/3D 시각화
  - Perplexity 조정 가능

### 6. 상관관계 분석
- 특징 간 상관관계 히트맵
- 처음 20개 특징 표시

### 7. 악성 심각도 분석 (선택 기능)
- 사이드바의 "🔬 악성 유방암 데이터 집중적으로 살펴보기" 체크박스 선택
- 악성 데이터만을 대상으로 한 심각도 분석
- KMeans 클러스터링으로 고악성/저악성 구분
- 악성 심각도 예측 모델 및 Feature Importance

## 파일 구조

```
breast/
├── web_dashboard.py          # Flask 서버 메인 파일
├── templates/
│   └── dashboard.html        # HTML 템플릿 (CSS/JS 포함)
├── static/                   # 정적 파일 (필요시)
├── kr_data.csv              # 데이터 파일
├── team_logo.png            # 로고 이미지
└── requirements.txt         # 필요한 패키지 목록
```

## 주의사항

1. 데이터 파일 경로:
   - 기본적으로 `kr_data.csv` 또는 `data.csv` 파일을 찾습니다
   - 파일이 없으면 오류가 발생합니다

2. 로고 이미지:
   - `team_logo.png` 파일이 있으면 자동으로 표시됩니다
   - 없으면 텍스트 "BC-Viz"로 대체됩니다

3. 포트 변경:
   - 기본 포트: 5000
   - 포트 변경은 `web_dashboard.py`의 마지막 줄 수정:
   ```python
   app.run(debug=True, host='0.0.0.0', port=5000)
   ```

## 문제 해결

### 데이터 로드 오류
- `kr_data.csv` 또는 `data.csv` 파일이 현재 디렉토리에 있는지 확인
- 파일 인코딩이 UTF-8인지 확인

### 포트 이미 사용 중
- 다른 포트로 변경하거나 실행 중인 프로세스 종료

### 모듈을 찾을 수 없음
- `pip install -r requirements.txt`로 모든 패키지 설치 확인

## 개발 모드

디버그 모드로 실행 (자동 재로드):
```python
app.run(debug=True, host='0.0.0.0', port=5000)
```

프로덕션 모드 (Gunicorn 등 사용 권장):
```bash
gunicorn -w 4 -b 0.0.0.0:5000 web_dashboard:app
```
